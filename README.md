# FBM出荷自動化ツール

Amazon FBM（自己発送）の出荷業務を自動化する無料ブラウザツールです。

**🔗 ツール：https://ec-tools-jp.github.io/fbm-tool/**

## 機能

**STEP 1：** Amazonセラーセントラルの未出荷注文レポート（TSV）→ ヤマトB2クラウド取込CSV に変換

**STEP 2：** ヤマトB2クラウドの発行済みデータCSV → Amazonセラーセントラルへの出荷通知TXT に変換

## 特徴

- インストール不要・サーバー不要（単一HTMLファイル）
- 入力データは外部に送信されません（全処理ブラウザ内完結）
- 複数店舗対応（設定画面から店舗情報を登録）
- Shift-JIS の入出力に完全対応

## 動作環境

Chrome または Edge が必要です（Safari・Firefox 非対応）
